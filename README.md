# Kuber Bassi — Personal Website & Portfolio

A multi-dimensional digital space for **Kuber Bassi**, showcasing a dual Professional Identity as a **Creative Developer** and an **Artist**. This project features an immersive, cinematic experience designed to bridge the gap between technical precision and artistic expression.

## 🚀 Key Features

### 1. Developer Portfolio ("God Mode")
A "God Mode" interface for exploring technical projects and engineering expertise.
- **Cinematic BIOS Loader:** A multi-phase interactive intro sequence that sets the stage for the developer experience.
- **Cyber HUD System:** A custom-built heads-up display showing real-time system data, including latency, clock, and scroll coordinates.
- **GitHub-Enriched Project Deck:** Projects are dynamically enriched with live metadata from the GitHub API, ensuring a real-time showcase of contributions.
- **Interactive Particle Network:** A physics-based background particle system with mouse-repulsion and connection logic.

### 2. Music Portfolio
A high-fidelity space dedicated to a music catalogue, production work, and artistic presence.
- **3D Hero Interface:** Interactive 3D tilt effects for a high-fidelity visual experience.
- **Infinite Music Catalogue:** A horizontal-scrolling showcase of album art and streaming links.
- **YouTube Integration:** A curated video grid with a custom portal-based modal player.
- **Dynamic Stats:** Animated counters for streaming milestones and listener engagement.

## 🛠️ Technology Stack

- **Core:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Animation:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Observer) & [Framer Motion](https://www.framer.com/motion/)
- **Routing:** [React Router](https://reactrouter.com/) (Subdomain-based architecture)
- **Analytics:** Vercel Speed Insights
- **Deployment:** Vercel
